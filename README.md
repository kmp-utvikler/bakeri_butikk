Klient server butikk applikasjon for personlige datamaskin. Systemet er bygget for å demonstrere
nettverks-, multitråd- og gui programmering i python, istedenfor å være en webutviklingsteknologi. 
For dette formålet bruker applikasjonen innebygde python moduler som socket, threading og tkinter,
i tillegg bruker det json og csv modulene for håndtering av data.
