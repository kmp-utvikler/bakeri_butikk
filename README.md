Klient server butikk applikasjon for personlige datamaskin, det har en grafisk brukergrensesnitt for 
både klient og server. Systemet er bygget for å demonstrere nettverks-, multitråd- og gui programmering 
i python, istedenfor å være en webutviklingsteknologi. For dette formålet bruker applikasjonen innebygde 
python moduler som socket, threading og tkinter. Siden håndtering av data ikke er et av målene brukes 
json og csv modulene til å håndtere dette. Penger blir ikke påregnet i dette systemet.
